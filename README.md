# IT Diagnostic Tool 🛠️

**IT Diagnostic** คือเว็บแอปพลิเคชันเครื่องมือวินิจฉัยและแก้ปัญหาคอมพิวเตอร์เบื้องต้นอย่างมีระบบ รวบรวมวิธีการตรวจสอบตั้งแต่ระดับ Hardware, Software, Network ไปจนถึงคำสั่ง CMD ที่จำเป็น จบครบในหน้าเดียว

🌐 **ลิงก์เข้าใช้งาน:** [https://timeyn0.github.io/it-tool/](https://timeyn0.github.io/it-tool/)

## ✨ ฟีเจอร์หลัก (Features)

*   🔍 **วินิจฉัยอาการ (Symptom Wizard):** ค้นหาวิธีแก้ปัญหาแบบ Step-by-Step จากอาการที่พบบ่อย เช่น อินเทอร์เน็ตใช้ไม่ได้, คอมช้า, จอฟ้า (BSOD), เปิดไม่ติด หรือโปรแกรมแครช
*   🌐 **LAN / Internet:** ลำดับขั้นตอนการตรวจสอบปัญหาเครือข่าย ตั้งแต่สาย Physical, IP Address, Router ไปจนถึง DNS และวิธีแก้ปัญหา Wi-Fi
*   💻 **ฮาร์ดแวร์ (Hardware):** เช็กลิสต์ตรวจสอบอุปกรณ์หลัก (CPU, RAM, HDD/SSD, PSU) พร้อมแนะนำโปรแกรม Freeware ที่ควรมีติดเครื่อง (เช่น HWiNFO64, CrystalDiskInfo)
*   ⚙️ **ซอฟต์แวร์ (Software):** แนะนำวิธีการจัดการระบบปฏิบัติการ, Windows Update, การสแกนไวรัส และการซ่อมแซม System Files
*   💻 **รวบรวมคำสั่ง CMD (Command Reference):** รวมคำสั่ง Network, Windows Repair และ Disk & Storage พร้อมปุ่มกดคัดลอก (Copy) ไปวางใช้งานได้ทันที

## 🚀 การใช้งาน (Usage)

โปรเจกต์นี้เขียนด้วย HTML, CSS และ JavaScript ภายในไฟล์เดียว (`index.html`) ทำให้มีความเบาและใช้งานได้ง่ายมาก

1.  **ใช้งานออนไลน์:** เข้าไปที่ลิงก์ GitHub Pages ของโปรเจกต์นี้
2.  **ใช้งานออฟไลน์:** สามารถดาวน์โหลดไฟล์ `index.html` ไปเก็บไว้ในเครื่อง และดับเบิลคลิกเปิดใช้งานผ่าน Web Browser ได้ทันทีโดยไม่ต้องต่ออินเทอร์เน็ต

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

*   HTML5
*   CSS3 (Custom UI Design, Responsive)
*   Vanilla JavaScript
