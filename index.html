ตรวจสอบทีละจุดจากซ้ายไปขวา เพื่อหาต้นตอปัญหา

ขั้นที่ 1 — Physical
  • ✓ ไฟ LED บน NIC/Switch ติดสีเขียว?
  • ✓ สาย LAN เสียบแน่นทั้งสองด้าน?
  • ✓ ลองเปลี่ยนสาย / Port อื่นบน Switch
  • ✓ Router/Modem ไฟปกติ ไม่กะพริบแดง?
ขั้นที่ 2 — IP Address
  • ✓ รัน ipconfig ดู IP Address
  • ✓ IP เป็น 169.254.x.x = ไม่ได้รับ IP จาก DHCP
  • ✓ ตรวจ Default Gateway ถูกต้อง?
  • ✓ ลอง ping 127.0.0.1 ทดสอบ loopback
ขั้นที่ 3 — Router/Gateway
  • ✓ ping [IP ของ Router เช่น 192.168.1.1] ได้ไหม?
  • ✓ เข้า Admin Panel ของ Router ได้ไหม?
  • ✓ Restart Router/Modem และรอ 2 นาที
  • ✓ ตรวจสถานะ WAN บนหน้า Admin Router
ขั้นที่ 4 — Internet/DNS
  • ✓ ping 8.8.8.8 ทดสอบ Internet IP
  • ✓ ping google.com ทดสอบ DNS
  • ✓ ping IP ได้ แต่ ping ชื่อไม่ได้ = DNS เสีย
  • ✓ เปลี่ยน DNS เป็น 8.8.8.8 หรือ 1.1.1.1
ปัญหา Wi-Fi
  • ✓ Channel เปลี่ยน Wi-Fi Channel เป็น 1, 6 หรือ 11 (2.4GHz) หรือ 36, 40 (5GHz) แทน Auto
  • ✓ Interference ไมโครเวฟและบลูทูธรบกวนคลื่น 2.4GHz ได้ ลองใช้ 5GHz แทน
  • ✓ Driver อัปเดต Wi-Fi Driver ผ่าน Device Manager หรือเว็บผู้ผลิต
  • ✓ Fix รัน netsh winsock reset แล้ว Restart เครื่อง
CPU / อุณหภูมิ
  • ✓ ตรวจ Temp ด้วย HWMonitor หรือ CoreTemp
  • ✓ CPU สูงกว่า 90°C = พัดลมเสีย หรือ Thermal Paste แห้ง HOT
  • ✓ ทำความสะอาดฝุ่นในเครื่องและบน Heatsink
  • ✓ เปลี่ยน Thermal Paste ถ้าใช้มาเกิน 3 ปี
RAM
  • ✓ รัน Windows Memory Diagnostic (mdsched.exe)
  • ✓ ถอดแล้วเสียบใหม่ ทำความสะอาดหน้าสัมผัสทอง
  • ✓ ลองใส่แค่ 1 แถวก่อน สลับ Slot ทดสอบ
  • ✓ ใช้ MemTest86 ทดสอบเต็มรูปแบบ
HDD / SSD
  • ✓ ตรวจสุขภาพด้วย CrystalDiskInfo
  • ✓ สถานะ Caution หรือ Bad = ใกล้เสีย สำรองข้อมูลด่วน!
  • ✓ รัน chkdsk C: /f /r ตรวจ Error
  • ✓ SSD ไม่ควรใช้พื้นที่เกิน 80% เต็ม
Power Supply (PSU)
  • ✓ เครื่องดับเอง/Restart บ่อย อาจเป็น PSU จ่ายไฟไม่พอ
  • ✓ ตรวจ Voltage ด้วย HWMonitor (+12V ควรอยู่ที่ 11.4–12.6V)
  • ✓ ตรวจว่า Watt ของ PSU เพียงพอกับ GPU รุ่นใหม่ไหม
  • ✓ ลองใช้ PSU สำรองทดสอบถ้าสงสัย
เครื่องมือ Freeware ที่แนะนำ
HWiNFO64 CrystalDiskInfo MemTest86 CPU-Z GPU-Z CoreTemp Malwarebytes ทุกโปรแกรม ฟรี 100%
Windows Update / Driver
  • ✓ อัปเดต Windows ให้ล่าสุดผ่าน Settings → Update
  • ✓ อัปเดต Driver การ์ดจอจากเว็บ NVIDIA / AMD โดยตรง
  • ✓ ตรวจ Device Manager หา ! สัญลักษณ์สีเหลือง
  • ✓ Rollback Driver ถ้าปัญหาเกิดหลังอัปเดตครั้งล่าสุด
Virus / Malware
  • ✓ สแกนด้วย Windows Defender → Full Scan
  • ✓ ใช้ Malwarebytes สแกนเสริมหลัง Defender
  • ✓ ตรวจ Startup ใน Task Manager → แท็บ Startup
  • ✓ ติดตั้ง Antivirus 3rd Party ซ้อนกันหลายตัวอย่าทำ
System Repair
  • ✓ รัน sfc /scannow ตรวจและซ่อม System File
  • ✓ รัน DISM RestoreHealth เพื่อซ่อม Windows Image
  • ✓ Boot ใน Safe Mode เพื่อ Isolate ว่าปัญหามาจากอะไร
  • ✓ ใช้ System Restore ถ้ามี Restore Point ที่ดี
Cleanup / Performance
  • ✓ Disk Cleanup หรือเปิด Storage Sense ล้างไฟล์ชั่วคราว
  • ✓ ปิดโปรแกรม Startup ที่ไม่จำเป็นใน Task Manager
  • ✓ ตรวจ Task Manager ว่า CPU/RAM/Disk ตัวไหน 100%
  • ✓ HDD ลอง Defrag ได้ (SSD ไม่ต้องทำ เป็นอันตราย)

Admin คำสั่งที่มีไอคอนนี้ต้องรัน CMD ในฐานะ Administrator (คลิกขวา → Run as administrator)
